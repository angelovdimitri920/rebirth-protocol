**Known issues to resolve first, before Stage 1 gameplay work:**
1. **`CombatantHealth.cs` design discrepancy.** It currently models: HP hits zero → `KnockedDown`, with a finite `RebirthCharges` count spent per rebirth attempt (destroyed once exhausted). `GAME_DESIGN.md` §2.2 — the mechanic the game is *named after* — describes something different: a hit drains both HP and a separate endurance pool at once; **endurance** emptying (not HP) triggers knockdown; rebirth invincibility is unlimited, not charge-limited. Redesign `CombatantHealth.cs` to match `GAME_DESIGN.md` §2.2 (the documented source of truth), update `CombatantHealthTests.cs` to cover the new shape including the previously-untested "already knocked down" and "recover stagger" paths, and note in your check-in that this was a deliberate correction, not a new decision you invented.
2. **`RebirthProjectSetup.EnsureBootstrapScene()` unconditionally regenerates `Bootstrap.unity` from an empty scene every time `ConfigureProject` runs** — and `scripts/unity-compile.ps1` calls `ConfigureProject` as its routine compile check. Fix so it only builds the scene if it doesn't already exist (or splits scene-authoring out of the routine compile step entirely) — otherwise any manual scene work gets silently wiped the next time someone runs the compile script.
3. **`Invoke-UnityBatch.ps1` hardcodes `D:\Unity\Hub\Editor\6000.3.19f1\Editor\Unity.exe`.** Make it a parameter or read from an environment variable, defaulting to that path so nothing breaks today.
4. **`RebirthProjectSetup.SetActiveInputHandling()` uses reflection** to set `PlayerSettings.activeInputHandling`, which is a directly-settable public property in this Unity version — replace the reflection with a direct assignment so a lookup failure can't silently no-op.
5. **`CLAUDE.md` at the repo root still describes the old Three.js/Rapier3D stack verbatim.** Rewrite it to describe the current Unity/C# stack, assembly layout, and working style (this file can otherwise follow the same shape/tone it already has).
